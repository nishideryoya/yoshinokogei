# 吉野工芸の里（WordPress 子テーマ）

石川県白山市・吉野工芸の里サイト用のカスタムテーマ `child-yoshinokogei` です。

## 必要な環境

- WordPress 6.x 以上
- PHP 8.x
- 親テーマ（bootstrap-basic4 系を想定）

## インストール

1. このフォルダを `wp-content/themes/child-yoshinokogei` に配置
2. 管理画面 → 外観 → テーマ で **child-yoshinokogei** を有効化
3. 管理画面に一度ログイン（固定ページ・施設データの自動作成）
4. **設定 → パーマリンク → 変更を保存**

## 主なページ

| URL（例） | 内容 |
|-----------|------|
| `/` | トップ |
| `/about/` | 吉野工芸の里について |
| `/guide/` | ご利用の案内 |
| `/map/` | 全体マップ |
| `/taiken/` | 体験教室 |
| `/news/` | お知らせ一覧 |

## 管理画面

- **施設** … マップ・トップの施設情報（共通データ）
- **体験教室** … 体験プログラム
- **作家** … 作家紹介
