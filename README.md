# 吉野工芸の里（WordPress 子テーマ）

石川県白山市・吉野工芸の里サイト用のカスタムテーマ `child-yoshinokogei` です。

## 必要な環境

- WordPress 6.x 以上
- PHP 8.x
- **Advanced Custom Fields（ACF）** プラグイン（必須）
- 親テーマ（bootstrap-basic4 系を想定）

## インストール

1. このフォルダを `wp-content/themes/child-yoshinokogei` に配置
2. 管理画面 → **プラグイン** で **Advanced Custom Fields** を有効化
3. 管理画面 → 外観 → テーマ で **child-yoshinokogei** を有効化
4. 管理画面に一度ログイン（固定ページ・施設データの自動作成）
5. **設定 → パーマリンク → 変更を保存**

## 職員向け：コンテンツの編集方法

サイトの文言・画像・一覧は、すべて WordPress 管理画面から編集できます。

### サイト共通（電話・住所・開館案内）

**サイト設定** メニュー

| 項目 | 内容 |
|------|------|
| サイト名 | ヘッダーの施設名 |
| ヘッダー下バー | 「10:00〜17:00」などの帯表示 |
| 電話・FAX・メール・住所 | 各ページのお問い合わせ欄に反映 |
| Instagram URL | トップのInstagramリンク |
| フッター文言 | コピーライト表示 |

### 各ページの編集

**固定ページ** から該当ページを開くと、ページ下部に ACF の入力欄が表示されます。

| ページ | 編集できる主な内容 |
|--------|-------------------|
| トップ | メインビジュアル、Aboutセクション、Instagram画像、News表示件数 |
| ご利用の案内 | 開館時間、料金表、体験料金目安、駐車場、アクセス、注意事項 |
| 体験教室 | リード文、予約セクション、Web予約URL |
| 吉野工芸の里について | ごあいさつ、3つの魅力、施設リスト、基本情報 |
| 全体マップ | キャッチコピー、リード文、ヒーロー画像 |

### 投稿・カスタム投稿

| メニュー | 用途 | 入力項目 |
|----------|------|----------|
| **投稿** | お知らせ・開催情報（News & Events） | タイトル、本文、アイキャッチ、カテゴリー（イベント/お知らせ/重要） |
| **体験教室** | 各体験プログラム | タイトル、本文、料金・時間・定員・対象・場所 |
| **施設** | マップ・トップの施設情報 | タイトル、説明、アイキャッチ、マップ上の位置（%） |
| **作家** | 作家紹介 | ジャンル、経歴、Instagram、作品画像 |

### カテゴリー（お知らせ）

- **イベント** … 開催情報・フェアなど
- **お知らせ** … 一般のお知らせ
- **重要** … 休館・変更など

## 主なページ URL

| URL（例） | 内容 |
|-----------|------|
| `/` | トップ |
| `/about/` | 吉野工芸の里について |
| `/guide/` | ご利用の案内 |
| `/map/` | 全体マップ |
| `/taiken/` | 体験教室 |
| `/news/` | お知らせ一覧 |

## ACF フィールドの管理

フィールド定義はテーマ内 `inc/acf/fields.php` にあり、編集内容は `acf-json/` に自動保存されます（Git 管理向け）。
